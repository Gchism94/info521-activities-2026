# D2L placeholder pages

One self-contained HTML page per weekly peer-engagement activity, for use as the
D2L assignment stub. Each page shows the four-stage loop, embeds the live Pages
tool in an iframe, and carries an "open in a new tab" escape hatch in case D2L's
frame is cramped or blocked.

**Usage (per assignment):** in D2L, either upload the matching `weekNN-*.html`
file as the assignment's content page, or create an HTML content page and paste
the file's source. Nothing else to configure — the pages are self-contained
(inline CSS, classic script conventions, no external dependencies beyond the
live tool).

**Term rollover:** the tool URLs point at the canonical Pages site
(`https://gchism94.github.io/info521-activities-2026/<slug>/`). If the base URL
changes, regenerate or search-replace it across the six files.

These pages are also served (harmlessly) on the Pages site under `/d2l/`; they
are not linked from the site's landing page.
