Bias, variance, and the cost of complexity

Each point is a patient's response at a given treatment dose; the true dose–response curve peaks at a moderate dose. Raise the polynomial degree to fit it and watch training vs. test error diverge, then use "Show 20 fits" to see variance directly and the ridge penalty $\lambda$ to tame it.

$$\mathbb{E}\big[(y - \hat f(\mathbf{x}))^2\big] = \underbrace{(\mathbb{E}[\hat f(\mathbf{x})] - f(\mathbf{x}))^2}_{\text{bias}^2} + \underbrace{\mathrm{Var}[\hat f(\mathbf{x})]}_{\text{variance}} + \sigma^2$$
ridge regression trades variance for bias
$$\hat{\mathbf{w}} = \arg\min_{\mathbf{w}} \lVert \mathbf{y} - \mathbf{X}\mathbf{w} \rVert^2 + \lambda \lVert \mathbf{w} \rVert^2$$
