K-means: discovering patient subtypes

Each point is a patient — BMI on the x-axis, fasting glucose on the y-axis, with no diagnosis labels. Drag the k centroids to seed them, then step through the assignment and update phases (or run to convergence) and watch clusters emerge. Try "New centroids" to see how the starting point changes where it lands.

$$\mathcal{L} = \sum_{k=1}^{K} \sum_{n:\, z_n = k} \lVert \mathbf{x}_n - \boldsymbol{\mu}_k \rVert^2$$
minimized by alternating assignment and update
$$z_n = \arg\min_k \lVert \mathbf{x}_n - \boldsymbol{\mu}_k \rVert^2 \qquad \boldsymbol{\mu}_k = \frac{1}{N_k}\sum_{n:\, z_n = k} \mathbf{x}_n$$
